运行bin\release\Hold'em.exe可以直接开始游戏
第一次启动时请务必确认控制台所使用的字体为点阵字体，否则会有大量图标无法显示以及一些不可预测的排版问题
设置方法：右键程序窗口顶部→属性→字体
